The machine has a pulse. Read it.

A model does not emit tokens evenly — it stalls on KV-cache pressure, bursts when the batch clears, jitters with the scheduler. Token EKG timestamps each token as it arrives and draws that hidden cadence as a live trace, with a spectral read of its rhythm. A real biosignal of the serving pipeline — not a mind.
