a glyph is a whole proposition

An invented language is a learned dictionary, not magic.

Type a sentence. A closed lexicon encodes it into circular logograms whose SHAPE carries the meaning — limbs for agency, a closed ring for a completed idea, weight for certainty. Hover a glyph and it blooms back into the proposition it holds — every glyph round-trips to exactly one meaning (a canonical decode; synonyms and function words collapse, so it recovers the proposition, not your wording). Feed it a word the lexicon never learned and watch it fall back to a marked base glyph — the honest failure of a closed vocabulary, shown not hidden.
